# 📝 Todo List

React + Redux로 만든 간단한 일정 관리 앱입니다.  
할 일 추가/삭제, 완료 체크, 검색, 다크모드를 지원하며 LocalStorage로 상태를 보존합니다.

🔗 [배포 사이트 바로가기](https://my-todolist-check.netlify.app/)

---

## ✨ 주요 기능

- Todo 추가/삭제, 완료 체크
- 검색(키워드 기반 필터링)
- 총 개수 / 완료 / 미완료 통계
- 다크/라이트 모드 전환
- 새로고침 후에도 목록·테마 유지 (LocalStorage)

---

## 🛠️ 사용 기술

- **React**
- **Redux**
- **JavaScript**
- **styled-components**
- **LocalStorage**
